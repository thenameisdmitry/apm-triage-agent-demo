APM Triage Agent, interactive simulator

When a payment method starts failing, someone has to answer three questions fast: is it real, how bad is it, and who should be woken up? This simulator shows how an AI agent answers them. Pick a scenario below and run the triage. All data is synthetic; the logic mirrors triage_agent.py and no model calls are made.
